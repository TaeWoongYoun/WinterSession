## 객체지향개발론 (이은주 교수님)

### 본 수업은 C++ 객체지향 프로그래밍과 관련된 개념을 이해할 수 있도록 한다. 클래스, 상속, 다형성, 연산자 오버로딩, 템플릿등을 학습하며 프로그래밍 과제를 통한 개념을 구체화한다. 또한, 문제 해결을 위한 사고 능력을 배양하고, 공학적 문제에 대한 해결 방법을 배운다.
