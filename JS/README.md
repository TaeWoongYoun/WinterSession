## 웹애플리케이션개발 (조상욱 교수님)

###  과목은 웹의 개념과 원리 그리고 웹 사이트를 제작하는데 필요한 HTML5, CSS3 그리고 자바스크립트를 단계별로 학습하고, 직접 웹사이트를 구현해 본다. 또한 웹 애플리케이션 개발에 필요한 웹 기반 기술, 다이내믹한 웹을 개발하는데 사용하는 응용 라이브러리인 제이 쿼리도 학습한다
